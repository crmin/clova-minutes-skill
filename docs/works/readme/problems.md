# Problems

## 문제 정의

저장소에는 영문 `SKILL.md`와 영문 `INSTALL.md`만 있었고, 사람이 처음 저장소를 열었을 때 스킬의 목적, 사용 시나리오, 결과물 특성을 빠르게 이해할 수 있는 소개 문서가 없었다.

## 원인

- 스킬 본문은 에이전트 실행 규칙 중심 문서다.
- 설치 문서는 설치 자동화와 경로 안내 중심 문서다.
- 사람에게 필요한 배경 설명과 예시는 별도 문서가 필요했다.

## 해결 내용

- 한국어 `README.md`를 새로 작성해 저장소 목적, 설치 시작 방법, 입력, 핵심 동작, 검증 가능성, 예시 요청, 샘플 파일을 정리했다.
- README 상단에 `INSTALL.md` URL을 포함한 영문 복붙용 설치 요청 문구를 배치했다.

## 회귀 방지

- 향후 스킬 동작이나 설치 정책이 바뀌면 `README.md`의 설명과 링크도 함께 갱신한다.
